# 🏥 PatienLens Dashboard

A modern **healthcare analytics dashboard** built using **HTML, CSS, JavaScript, and Chart.js**.
This project fetches real-time patient data from an API and visualizes key health metrics in an interactive UI.

---

## 🚀 Features

* 🔐 **API Integration with Authentication (Basic Auth)**
* 📋 **Dynamic Patient List (Sidebar)**
* 👤 **Patient Details View**
* 📊 **Blood Pressure Chart (Chart.js)**
* ❤️ **Health Metrics (Heart Rate, BP, Status)**
* 🎯 **Default Patient Selection**
* 🎨 **Modern SaaS Dashboard UI**
* 📱 **Responsive Design**

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Chart Library:** Chart.js
* **API:** Coalition Technologies Patient API

   ```

   ```

3. Run the project:

## 📊 How It Works

1. Fetch patient data from API
2. Display patients in sidebar
3. Click a patient to:

   * View details
   * See latest health stats
   * Visualize BP trends in chart
